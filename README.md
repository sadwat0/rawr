# RAWR (Realistic Aquatic Warfare Representation)
Война... Война никогда не меняется... Множество обитателей моря вышло на сушу, и стало понятно, что территории не хватит на всех. Еще вчера вы были обычной водоплавающей рыбой, а сегодня вам предстоит сражаться за то, чтобы построить светлое будущее для своего народа на маленьком острове. Для этого нужно выгнать с острова всех остальных обитателей.

## Игровой процесс
Игра рассчитана на 2-4 игрока. Каждому игроку изначально дана одна клетка в его расположении, а вокруг - пустые, никем не заселенные клетки. Кликом на клетку, соседнюю с вашей, можно увеличить ее силу, если это ваша клетка, и уменьшить, если не ваша. Когда сила клетки падает ниже нуля в результате клика, вы становитесь ее владельцем. Когда у игрока не остается его клеток, он выбывает. Побеждает последний оставшийся игрок.

Кликать можно с интервалом в 0.5 секунд. Также в игре есть золото: выдается по 1 золоту в секунду плюс количество захваченных клеток-шахт. За золото можно покупать различные бонусы.

Для того чтобы начать, нужно создать либо присоединиться к лобби. Хост может по своему желанию начать игру.

## Управление
- WASD - движение камеры
- 1-7 - покупка бонусов за золото
- Клик мыши - атака по чужой клетке или усиление своей

## Мультиплеер
Мультиплеер реализован в локальной сети - нужно иметь два или более компьютера в локальной сети с открытыми портами в фаерволле. Один из компьютеров создает лобби, остальные к нему присоединяются. Хост может начать игру в лобби в любой момент.
